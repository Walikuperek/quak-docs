## Description

CreativoApp is an app for managing orders and parts in a wholesaler company. It is a tool for managing clients, orders, and parts. It is also a tool for generating reports and statistics.

> TechApp is very similar to CreativoApp. The difference is that TechApp is for a different company and is much more complex.
>
> [TechApp](/portfolio/tech-app) - for a company that repairs electronics:
>   - managing clients, tickets, and parts
>   - integrated with maps
>   - allows to sign documents in app

Wholesaler data about clients, orders, and parts is exported as XLSX(excel file) and then imported into CreativoApp. The app allows users to search for clients, orders, and parts. It also allows users to edit and delete clients, orders, and parts.

App allows to generate PDF invoices.

### Desktop and mobile

Data is inserted with the same app but in wholesaler office (installable desktop version as well).

It is a PWA application (can be installed on any Desktop and any iOS/Android).
