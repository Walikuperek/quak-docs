## Description

Qgames is a free games search engine. Database is based on [FREETOGAME](https://www.freetogame.com/) API.

### Desktop and Mobile

This app is responsive and works on both desktop and mobile devices.

### Technologies

Frontend is built with `Angular`, backend is built with `Node.js` and `Express.js` + `MySQL` as a database.

### Languages

Language: `only English`.
