## Description

Qwallet is an app for orginizating your finances. It is a simple and easy-to-use application that helps you to manage your money.

### Desktop

It is a PWA application (can be installed on any Desktop and any iOS/Android).

### Technologies

Frontend is built with `Angular` and `Node.js` with `Nest.js` as a backend.

### Languages

Language: `only English`.
