## Description

ShopApp ment to be a simple e-commerce app but installable(PWA) on clients phones, so they can use it quicker and easier. It was ment to create with customers greater bond.

ShopApp is also a web application that allows you to manage your store. It is a simple and intuitive tool that allows you to add, edit and delete products and manage orders from customers.

### Desktop and mobile

ShopApp is a web application that can be used on both desktop and mobile devices.
