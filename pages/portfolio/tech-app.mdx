## Description

TechApp is an app for managing tickets and generating documents after a repair (customers approve that job was done by sign in app). Mainly it is used by technicians in the field. TechApp allows to open navigation straight to the customer's address.

### Desktop and mobile

Data is inserted with the same app but in office (installable desktop version as well).

It is a PWA application (can be installed on any Desktop and any iOS/Android).

### Technologies

Frontend is built with Angular. The backend is built with Node and Nest.js framework.

### Languages

Language: only Polish.
