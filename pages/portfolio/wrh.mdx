## Description

Wielkopolski Rajd Historyczny is a website for a rally event that took place in 2018-2020. The website was created to provide information about the event, its history and the route of the rally. The website also contains a gallery with photos from the event and a list of sponsors.

### Technologies

Only basic frontend technologies were used in this project like `HTML`, `JS` and `CSS`.

### Languages

Language: `only Polish`.
