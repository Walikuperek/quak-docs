## Description

Qcrypto is a cryptocurrency values tracking application. It allows you to track the value of your cryptocurrency in almost real-time (Coingecko for free had 1-15min delay).

### Desktop and Mobile

This app is responsive and works on both desktop and mobile devices.

### Technologies

Frontend is built with `Angular`, backend is built with `Node.js`, `Express` library and `Coingecko` API client.

### Languages

Language: `only English`.

### Currencies

Plenty of cryptocurrencies are supported, including `Bitcoin`, `Ethereum`, `Litecoin`, `Ripple`, `Dogecoin`, `Cardano`, etc.

> Up to **250 most common cryptocurrencies** are supported.
