## Description

Qchat is a chat application for companies that want to provide a better customer service or as a secret messaging app in organization. Can be used as a chat for a website or as a standalone application.

### Desktop and Mobile

It is a PWA application (can be installed on any Desktop and any iOS/Android).

### Technologies

Frontend is built with `Angular` and `Firebase` as a backend.

### Languages

Language: `only Polish`.
