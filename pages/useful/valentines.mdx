# Valentines Card

## Description

This is a simple valentines card that I made for my girlfriend. It's a simple web page that you can show to your valentine.

## How to use

> Enter website [quak.com.pl/valentines](https://quak.com.pl/valentines/)

But it will display

```
Happy Valentines Day

<Heart animation>

Kali
```

![Screenshot](/assets/useful/valentines_kali.png)

## How to change name

Enter website:

```
https://quak.com.pl/valentines/?name=YOUR_NAME

with spaces:
https://quak.com.pl/valentines/?name=YOUR%20NAME
```

Effect:

![Screenshot](/assets/useful/valentines_custom_name.png)

That's all. Enjoy!
