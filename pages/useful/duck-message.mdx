# Duck Message Card

## Description

This is a simple message card. It's a simple web page that you can show to anyone with a message on it.

## How to use

> Enter website [quak.com.pl/duck_walk](https://quak.com.pl/duck_walk/)

But it will display

```
<walking duck animation>

<Text Here>
```

![Screenshot](/assets/useful/duck_walk.png)

## How to change text

Enter website:

```
https://quak.com.pl/duck_walk/?text=Your_text

with spaces:
https://quak.com.pl/duck_walk/?text=Your%20text
```

Effect:

![Screenshot](/assets/useful/duck_walk__text_here.png)

That's all. Enjoy!
